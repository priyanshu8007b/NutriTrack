# **App Name**: PaushtikPath

## Core Features:

- Secure User Authentication & Profile: Enable users to register, log in, and manage their basic profile information, interacting with the Django backend for secure credential handling.
- Indian Meal Logging Interface: Provide an intuitive interface for users to log Indian meals and specify quantities, displaying real-time calorie and macro calculations sourced from the Django backend.
- Smart Indian Meal Suggestion Tool: A generative AI tool that analyzes logged meals against daily nutritional goals and suggests culturally relevant Indian dishes or meal combinations to help meet remaining macro targets.
- Personalized Goal Management: Allow users to set and modify daily calorie targets and macro distribution ratios (protein, carbs, fats), with preferences persisted via the Django backend.
- Interactive Analytics Dashboard: Display visual breakdowns of daily calorie intake, macro distribution, weekly trend graphs, and goal progress, dynamically fetched and processed from the Django/Flask backend.
- Indian Food Database Browser: Allow users to search and explore a comprehensive database of Indian food items with their detailed nutritional information to facilitate accurate meal logging.

## Style Guidelines:

- Primary color: Rich mustard (#C68C09) to represent the warmth and earthy tones of Indian spices and wholesome food.
- Background color: Soft, warm off-white (#F9F3E7) to create a clean, fresh, and health-focused canvas.
- Accent color: Vibrant terracotta (#F2651A) for calls-to-action and highlighting key interactive elements, reminiscent of traditional Indian pottery and artistry.
- Unified font: 'Inter' (sans-serif) will be used for both headlines and body text, ensuring modern readability, neutrality, and clear presentation of nutritional data.
- Utilize clean, minimalist line icons for navigation and data visualization, subtly complemented by geometric patterns or stylized motifs inspired by traditional Indian art for decorative accents.
- Employ a spacious, card-based layout with clear visual hierarchy, optimized for quick meal logging, intuitive dashboard navigation, and responsiveness across all device types.
- Incorporate smooth and subtle animations for loading states, chart transitions, and form interactions to enhance user feedback and contribute to an overall fluid and polished user experience.
